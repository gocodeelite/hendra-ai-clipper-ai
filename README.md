# Jalankan dan Deploy Aplikasi Anda

Ini berisi semua yang Anda perlukan untuk menjalankan aplikasi Anda secara lokal.

## Jalankan Secara Lokal

**Prasyarat:** Node.js

1.  **Instal Dependensi:**
    `npm install`
2.  **Siapkan Kunci API:**
    Aturdi file **`.env.local`** dengan kunci API Anda.
3.  **Jalankan Aplikasi:**
    `npm run dev`# clipper
